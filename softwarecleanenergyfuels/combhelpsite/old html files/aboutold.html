Software for Clean Energy Fuels was created to make a code that represents many of the
capabilities of the Chemkin chemical kinetics package. The purpose of the Matlab code
was to avoid Chemkin fees and also to make the code much easier to understand by using
a high-level programming language. A GNU Octave-Fortran Hybrid version was created
that is still very easy to understand and avoids both Chemkin and Matlab fees.
